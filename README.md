# 🏦 Bank Management System (GUI + SQLite)

A fully functional **Bank Management System** built using **Python, Tkinter (ttkbootstrap)** and **SQLite**, designed with a modern UI and real-world banking features.

---

## 🚀 Features

### 👨‍💼 Admin Panel
- Secure Admin Login
- Open Customer Account
- View Customer Details
- Manage Transactions
- Card Management
- Loan Services

### 👤 Customer Panel
- Customer Registration with Validation
- Secure Login Authentication
- View Account Details
- Transaction History
- Account Statement
- Card & Loan Information

---

## 🎨 UI Highlights
- Modern aesthetic UI using **ttkbootstrap**
- Separate themes for:
  - Admin Panel 🟥
  - Customer Panel 🟦
- Center-aligned responsive layout
- Smooth window transitions
- Clean and professional design

---

## 🔐 Validations Implemented
- ✅ Email format validation
- ✅ Aadhar: 12-digit numeric
- ✅ Phone: 10-digit numeric
- ✅ Required field checks

---

## 💾 Database (SQLite)
- Admin Table
- Customer Table
- Transactions Table
- Cards Table
- Loans Table

---

## 🛠️ Tech Stack

- **Language:** Python 🐍
- **GUI:** Tkinter + ttkbootstrap
- **Database:** SQLite
- **IDE:** VS Code
